# 📁 discord-clone

**딥다이브 프론트엔드 3기 | 4월 팀스터디 활동**

해당 프로젝트는 팀 스터디 중 진행한 디스코드 클론 UI 구현 결과물입니다.  
주요 목표는 디스코드의 UI/UX 구조를 직접 구현해보며 **React에 대한 실전 감각**을 익히는 것이었습니다.
